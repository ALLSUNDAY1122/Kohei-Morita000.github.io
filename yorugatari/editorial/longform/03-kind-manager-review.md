## 起承転結の確認

### 起

- 大学進学、一人暮らし、学費不足をアルバイト継続の現実的な理由として設定
- 店長が失敗を責めず、食事や送迎を提供する実際の親切を描写
- 退職者の住所を覚えていること、安全確認カード、卒業生当番を初期の違和感として配置
- 主人公が店長へ恩を感じるため、異常を即座に悪意と断定できない状態を作った

### 承

- 友人名、授業時間、帰宅経路が複数人から店長へ集まる仕組みを段階的に開示
- 帰宅経路を変えた直後の連絡、鍵の傷、匿名警告で監視の具体性を強めた
- 元従業員・榊の証言により、恩義、借金、秘密によって協力者の立場が異なることを示した
- ノート、当番表、合鍵から、店長個人ではなく地域別の監視網であることを確定した

### 転

- 架空の外出予定によって、同僚から卒業生へ情報が流れることを主人公自身が検証
- 退職を告げても「見守り」は終わらないという店長の論理を提示
- 店長が勤務中である時刻に、元従業員が合鍵で室内へ侵入している展開へ反転
- レシート、食事、合鍵、クローゼット内の協力者により短編版の核を拡張して回収
- 主人公が廊下へ逃げて近隣住民へ助けを求め、受動的な被害者だけで終わらない構成にした

### 結

- 警察と運営会社が介入しても、任意の情報共有を含む監視網の全容は把握できない
- 転居、番号変更、転職で一度は日常を回復させる
- 新しい職場へ安全確認カードと監視情報が引き継がれていることを開示
- 最後は新しい同僚も協力者だった可能性を示し、「優しさ」が感染・継承される形で終えた

## 伏線回収

| 伏線 | 回収状況 |
|---|---|
| 店長が元従業員の住所を覚えている | 退職後も生活情報を更新している監視網として回収 |
| 安全確認カードの空欄を店長が知る | 同僚からの情報共有が常態化していると判明 |
| 卒業生当番に地域名がある | 元従業員が地域ごとに対象者を確認する当番表として回収 |
| 鍵に増えた傷 | 勤務中に持ち出され、合鍵が作成された可能性として回収 |
| 匿名の「鍵を持たせないで」 | 榊が過去の経験から送った警告として回収 |
| ノートの「継続」「再確認」「終了」 | 対象者の追跡状態を示し、結末の安全確認カードへ接続 |
| 店長が食事を与える親切 | 室内侵入と生活管理を正当化する行為へ反転 |
| 店長が勤務中のレシート | 店長以外の協力者が侵入した証拠として回収 |

## 文章監査

- 店長を最初から露骨な悪人にせず、実際に従業員を助けた事実と歪んだ使命感を両立させた
- 主人公が店をすぐ辞めない理由を、金銭事情、孤立、店長への恩義で補強した
- 監視者を単純な共犯集団にせず、善意、依存、恐怖、無自覚によって参加動機を分けた
- 合鍵作成や監視の具体的手順は詳述せず、結果と心理的侵害を中心にした
- 前2作品のような超常現象を使わず、最後まで人間の組織行動だけで恐怖を成立させた
- 店舗、大学、帰宅路、自宅、新しい職場へ舞台を移し、同一場面の反復を避けた
- 作品形式を意識させるメタ表現を削除し、主人公の心理表現へ修正した
